The : Technique

  • : 表示所有
  • 2: 表示从右往左取2个
  • :5 表示从左向右取5个

For example:

import numpy as np

a = np.array([[1, 2, 3], [0, 0, 0], [2, 3, 4]])

b = a[2:, :3] # Works only for numpy array
print(b)
