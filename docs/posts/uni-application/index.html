This post only focuses on CS / ECE MS programs.

1 Notes

1.1 General

  • 一般申请入口都在 Admission 或 About 页面中就有, 选择 Graduate Admission, 然后一般会叫你选择一个 program (就是专业).

  • 申请尽量 as early as possible, 有些学校申请满了就会关闭入口!!

1.2 English

  • IELTS 成绩都要求两年内.

  • GRE 都要求 5 年内.

1.3 申请材料

  • SoP (Statement of Purpose): 个人陈述, 介绍自己为什么申请这个专业, 以及自己的研究兴趣等.
    • SoP 一般都要求 1-2 页, 有字数限制.
  • CV (Curriculum Vitae): 简历
    • 有些学校需要有些不需要
  • Transcripts: (英文) 成绩单.
    • 大部分都需要认证 (Credential Evaluation)! 一般是 WES 或 ECE (Educational Credential Evaluators).
  • LOR (Letter of Recommendation): 推荐信, 一般需要 3 封.

3 Terms Explanation

Below is some terms that you may encounter when applying.

  • ACT / SAT: American College Test / Scholastic Assessment Test, 美国高考, 与我们无关

  • MS / MSE: 研究性硕士, MEng: 课程型.

  • BS / MSE: 本硕连读 5 年制的项目, 一般不对留学生开放, 注意这个跟 MSE 不一样!

  • Fall / Spring Admission: 一般都是 Fall 入学.
